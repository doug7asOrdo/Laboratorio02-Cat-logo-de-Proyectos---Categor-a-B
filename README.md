# Laboratorio 2 - ISW-521
## Categoría B: Interfaces Interactivas y DOM Avanzado

## Descripción

Este proyecto corresponde al Laboratorio #2 del curso **ISW-521 - Programación en Ambiente Web I** de la Universidad Técnica Nacional.

El objetivo del laboratorio es desarrollar una aplicación web interactiva utilizando JavaScript , enfocándose en la manipulación avanzada del DOM, consumo de una API REST mediante Fetch, manejo de autenticación JWT, programación asíncrona con `async/await` y construcción de interfaces resilientes ante errores de red.

---

## Objetivos

- Consumir la API REST pública del Mundial 2026.
- Manipular el DOM de forma dinámica.
- Implementar autenticación mediante JWT.
- Utilizar únicamente `async/await` para llamadas asíncronas.
- Implementar manejo de errores HTTP.
- Aplicar estrategias de resiliencia en la interfaz.

---

##  Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- LocalStorage
- API REST World Cup 2026

---

## Funcionalidades generales del laboratorio

El proyecto implementa las siguientes características generales:

- Autenticación mediante JWT.
- Consumo de servicios REST.
- Manejo de errores HTTP (401, 429 y 500).
- Reintentos automáticos mediante Backoff Exponencial.
- Modo Offline utilizando LocalStorage.
- Interfaz completamente dinámica mediante manipulación del DOM.
